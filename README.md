# Todo App (Vanilla JavaScript)

A modern and minimal todo application built with **pure JavaScript**, focused on clean logic, state-driven UI, and DOM manipulation without any frameworks.

Live demo:  
https://amirhosseinln.github.io/Todo-app-V1/

---

## Features

- Add todos using button click or Enter key
- Mark tasks as completed / uncompleted
- Delete tasks
- Live task summary:
  - Total tasks
  - Completed tasks
  - Remaining tasks
- Persistent data using localStorage
- Smooth UI animations
- Dark, minimal user interface

---

## JavaScript Concepts Applied

- State management with arrays
- Event listeners
- Conditional rendering
- Dynamic DOM creation
- Separation of logic and rendering
- Local storage persistence

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
